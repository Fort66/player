МУЛЬТИМЕДИА ПЛЕЕР С ПУЛЬТОМ ЦВЕТОМУЗЫКИ

Эта программа реализована в рамках обучения моих студентов созданию GUI на PySide6 и доп. возможностям замечательного фреймворка QT.

Огромная благодарность разработчикам QT и PySide6 за возможность делать чудесные вещи!

Весь код подробно документирован, поэтому не вижу смысла описывать программу здесь.

Цветомузыка включается (отключается) по кнопке с цветным кругом. Меню управления цветомузыки вызывается по правой кнопке мышки в окне.
